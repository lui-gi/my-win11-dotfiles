## Overview

- **[YASB](https://github.com/amnweb/yasb)**: status bar, mimics Waybar

- **[komorebi](https://github.com/LGUG2Z/komorebi)**: tiling window manager for Windows 11

- **[Windhawk](https://windhawk.net/)**: Windows 11 tweak/mod manager

- **[ExplorerBlurMica](https://github.com/Maplespe/ExplorerBlurMica)**: applies mica blur efffects to File Explorer
